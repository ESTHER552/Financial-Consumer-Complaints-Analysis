# Financial Consumer Complaints Analysis Dashboard
### Excel Capstone Project

---

## Project Overview

I took on the role of a Data Analyst Intern for **Cumulus Financial**, a simulated financial services firm seeking deeper insights into consumer complaints filed against the organization.

The goal was to design an interactive **Financial Consumer Complaints Analysis Dashboard** in Microsoft Excel to identify complaint trends, assess resolution efficiency, and support data-driven service improvement strategies.

---

## Problem Statement

Cumulus Financial was experiencing challenges in understanding:
- Which products and issues were generating the most complaints
- Why certain complaints went unresolved or were disputed by consumers
- Whether response timeliness correlated with consumer satisfaction outcomes
- How to allocate resources more effectively for complaint resolution

---

## Dataset

The dataset spans three normalized tables:

| Table | Description |
|-------|-------------|
| `Complaints` | Core fact table - submission channel, state, resolution outcome, dispute status, timeliness |
| `Products` | Dimension table - financial product and sub-product categories |
| `Issues` | Dimension table - primary issues and sub-issues per complaint |

---

## Tools Used

- **Microsoft Excel** - data cleaning, transformation, analysis, and dashboard

---

## Project Workflow

### 1. Data Cleaning & Transformation
- Handled missing and inconsistent values across all three tables
- Created calculated columns to enrich the dataset (e.g. response time, dispute flags)
- Normalized and merged tables into a single cleaned analysis-ready table
- Incorporated reference data to make abbreviated fields (e.g. state codes) more meaningful

### 2. Data Analysis
- Built pivot tables to explore complaint volume by product, issue, channel, and state
- Analyzed the relationship between company response type and consumer dispute rates
- Identified patterns in unresolved and disputed complaints
- Assessed timeliness of responses across segments

### 3. Dashboard Development
- Designed an interactive Excel dashboard with:
  - **KPI cards** - total complaints, dispute rate, timely response rate
  - **Charts** - complaint trends over time, top products, top issues, resolution breakdown
  - **Slicers** - dynamic filtering by product, channel, state, and date
